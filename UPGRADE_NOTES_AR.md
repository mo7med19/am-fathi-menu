# ملخص تطوير المنيو الإلكتروني

هذه النسخة تعمل بدون باك إند، وتعتمد على ملف `src/data/menu.json` كمصدر البيانات الأساسي، وترسل الطلب النهائي إلى واتساب من خلال رابط `wa.me`.

## أهم ما تم تطويره

1. إصلاح بنية السلة، السعر أصبح محفوظًا باسم `unitPrice` ويتم حساب الإجمالي منه بشكل صحيح.
2. استخدام رقم واتساب من ملف `menu.json` بدل الرقم الثابت داخل الكود.
3. إنشاء رسالة واتساب منظمة تشمل بيانات العميل، تفاصيل الأصناف، الكمية، سعر الوحدة، الإجمالي، والملاحظات.
4. إضافة حفظ السلة في `localStorage` حتى لا تضيع عند تحديث الصفحة.
5. تحسين واجهة السلة، تعديل الكمية، حذف صنف، مسح السلة.
6. إضافة نموذج بيانات العميل داخل السلة، الاسم ورقم الموبايل إلزاميان، والعنوان والملاحظات اختياريان.
7. تحسين البحث، البحث يعمل على الاسم والوصف والتاجات، وإخفاء الأقسام الفارغة.
8. استخدام ألوان الثيم من ملف `menu.json` تلقائيًا.
9. إضافة صورة بديلة للأصناف التي لا تحتوي على صورة.
10. تحسين التوافق مع الموبايل لأن الاستخدام الأساسي غالبًا سيكون من QR أو رابط واتساب.

## طريقة تعديل بيانات المطعم

افتح الملف:

`src/data/menu.json`

ثم عدل القيم التالية حسب الحاجة:

```json
{
  "restaurantName": "اسم المطعم",
  "restaurantType": "نوع المطعم",
  "description": "وصف مختصر",
  "whatsapp": "201XXXXXXXXX",
  "phone": "+20 1XXXXXXXXX",
  "currency": "جنيه"
}
```

رقم واتساب يجب أن يكون بصيغة دولية بدون علامة +، مثل:

`201118921117`

## تشغيل المشروع

```bash
npm install
npm run dev
```

## بناء نسخة نهائية للنشر

```bash
npm run build
```

بعدها استخدم مجلد `dist` للنشر على أي استضافة Static Hosting مثل Netlify أو Vercel أو GitHub Pages.


## إصلاح توافق الموبايل V2

تمت إضافة قواعد CSS لمنع الـ horizontal overflow وتحسين العرض على شاشات الموبايل:

- ضبط `html`, `body`, و `#root` على `width: 100%` و `overflow-x: hidden`.
- تحويل هيدر الموبايل إلى layout عمودي بدل flex أفقي.
- إجبار النصوص الطويلة مثل رقم الهاتف والوصف على الكسر داخل حدود الشاشة.
- استخدام `minmax(0, 1fr)` في الـ grid حتى لا تتسبب الكروت في توسيع الصفحة.
- تحسين ارتفاع الصور وأبعاد الكروت على الشاشات الصغيرة.
- نقل درج السلة ليظهر من اليمين بما يناسب واجهة RTL العربية.
- إضافة دعم `viewport-fit=cover` و `dir=rtl` في ملف `index.html`.
